Box-model

Box-model is a combination of different properties which gives a receptacle of a web. People use it to design a conformation of web or blog. It is useful for designing a web because browser able to open the web (every single element in the web) as shape of its designer wanted to be by box-model. There are quite many properties contained the box-model, these are only few of that:

  • Margin-bottom
  • Border
  • Padding
  • 'Content-box' : content-box sets the default of css box-model shape. The content-box adjusted every size of element which contained the document what is more the other property add the size to make the element wider than it was.
  • 'Padding-box' : Padding-box is the area which has nothing inside (transparent) it is always around the content.
  • 'Border-box' : Border-box sets the record of any values from border and padding which the elements sets by the designer to apply in the browser.
