The default value of each property :

  • table-layout: auto
  • border-spacing: 2px
  • border-collapse: separate
  • caption-side: top
  • empty-cells: show

The HTML element correspondant to these display value:

  • table value corresponds of <table> element
  • table-row-group value corresponds of <tbody> element
  • table-header-group value corresponds of <thead> element
  • table-footer-group value corresponds of <tfooter> element
  • table-row value corresponds of <tr> element
  • table-cell value corresponds of <td> element
  • table-column-group value corresponds of <colgroup> element
  • table-column value corresponds of <col> element
  • table-caption value corresponds of <caption> element
  1. Border-spacing can work effectively if only border-collapse is separate.
  2. Downside of using various table "table" value of display: we can not use tabular html, You miss the layout structure.

The difference between table-layout: fixed and table-layout: auto

  • Table-layout: auto sets automatically the table layout on the browser.
  • Table-layout: fixed sets the fixed table-layout.

Yes, I can use border-spacing because it sets the space between two cell in element but we have to display table.

Empty-cells work effectively only when border-collapse property is separate.
